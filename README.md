Pair Programing Modulo 1 - Python: Pair Programming y Ejercicios Prácticos

En este repositorio, exploraremos las prácticas de pair programming, realizadas para mejorar la colaboración y habilidades de codificación en equipo.

## Recopilación de Ejercicios Prácticos

Son una colección de ejercicios prácticos para ayudar a consolidar  los conocimientos en Python. Estos ejercicios abarcan temas como sentencias de control, funciones, clases y expresiones regulares.

### Contenido del Módulo:

1. **Sentencias de control**
   - Ejercicios que abordan el uso de estructuras como bucles for, list comprehension, o try...except.

2. **Funciones**
   - Actividades diseñadas para comprender la definición, uso y retorno de funciones en Python, incluyendo funciones con argumentos y funciones recursivas.

3. **Clases**
   - Desafíos centrados en el manejo de clases y objetos en Python, explorando la definición de clases, la herencia, los métodos y atributos de instancia, entre otros conceptos.

4. **Expresiones regulares**
   - Ejemplos y ejercicios para practicar el uso de expresiones regulares en Python, incluyendo la búsqueda, coincidencia y sustitución de patrones en cadenas de texto.
